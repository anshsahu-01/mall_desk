# ANIMATION SYSTEM

The animation system should feel cinematic and immersive.

Scrolling must feel deeply connected to the experience.

Use:
- GSAP ScrollTrigger
- pinned sections
- layered parallax
- smooth inertia
- cinematic reveals
- spatial transitions
- immersive transformations

Important:
Animations should not feel repetitive.

The experience should contain dynamic motion behavior.

Examples:
- paint lines drawing themselves while scrolling
- random animated strokes appearing across the viewport
- environmental motion reactions
- cinematic masking transitions
- media scaling dramatically
- giant typography moving through the screen
- products rotating subtly in 3D
- depth-based motion systems
- layered image movement
- horizontal cinematic galleries

Media behavior:
- videos should occupy large space
- many sections should use fullscreen visuals
- media should dynamically resize while scrolling
- cinematic zoom transitions should occur naturally

Three.js animations should:
- react to scroll position
- rotate subtly
- animate smoothly
- maintain premium realism

Motion should feel:
- expensive
- elegant
- restrained
- cinematic
- immersive

Avoid:
- playful bounce animations
- childish movement
- random floating effects
- overused particle spam
- excessive neon motion

The animation pacing should alternate between:
- calm
- dramatic
- immersive
- energetic

The experience should feel choreographed like a film sequence.

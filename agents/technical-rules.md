# TECHNICAL RULES

Framework:
- Next.js latest App Router
- TypeScript strict mode

Styling:
- Tailwind CSS only
- no inline styles unless absolutely necessary

Animation Stack:
- GSAP
- Framer Motion
- Lenis smooth scrolling
- ScrollTrigger
- React Three Fiber
- Drei

Performance Requirements:
- smooth scrolling
- GPU-friendly animations
- optimized rendering
- lazy loading
- optimized media loading
- no animation jank
- avoid memory leaks

Three.js Rules:
- use physically believable lighting
- use cinematic shadows
- use realistic materials
- use floating camera movement
- avoid childish 3D scenes
- avoid low-quality assets
- avoid cartoon aesthetics

Code Structure:
- reusable components
- modular sections
- scalable architecture
- proper file separation
- maintainable structure

Animation Rules:
- animations must feel smooth and premium
- avoid over-animation
- use cinematic pacing
- use easing carefully
- animation should support storytelling

Responsive Rules:
- desktop-first
- tablet optimized
- responsive scaling
- maintain immersive layouts

Asset Rules:
- if assets are unavailable, generate premium placeholders automatically
- use high-quality visuals only
- avoid low-resolution imagery

Do not generate:
- generic hero sections
- startup website layouts
- repetitive templates
- childish interactions
- cheap visual effects

Everything should feel premium and intentional.

# VISUAL LANGUAGE

The visual direction must feel premium, modern, cinematic, and architectural.

Primary visual inspirations:
- Apple
- Tesla
- Hermès
- Saint Laurent
- Sphere Las Vegas
- luxury flagship showrooms
- futuristic retail architecture

Use:
- large typography
- oversized layouts
- fullscreen imagery
- dramatic spacing
- cinematic framing
- immersive scale
- visual restraint

Avoid:
- clutter
- too many UI cards
- boxed layouts
- excessive borders
- cheap gradients
- random UI kits
- colorful startup aesthetics

Typography Rules:
- text should be large
- headlines should dominate the screen
- typography should often become part of the visual composition
- use spacing aggressively
- avoid long paragraphs
- use short impactful statements

The interface should rely on:
- black
- white
- subtle grayscale
- premium contrast
- occasional metallic accents

The design should feel:
- elegant
- confident
- futuristic
- high-budget

Sections should often contain:
- fullscreen media
- giant visuals
- oversized product renders
- cinematic compositions
- layered depth

Layouts should continuously evolve while scrolling.

Do not repeat the same section structure repeatedly.

The UI should feel unpredictable but controlled.

Some sections should:
- expand while scrolling
- collapse dynamically
- transform layout structure
- reveal hidden visuals
- use horizontal motion
- use pinned scroll storytelling

The interface should feel alive.

Whitespace should be used aggressively.

The user should feel immersed inside a digital cinematic environment rather than browsing a website.

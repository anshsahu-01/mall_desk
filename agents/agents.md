# PROJECT IDENTITY

This project is a cinematic interactive luxury sales experience built with Next.js.

It is NOT:
- a startup landing page
- a SaaS dashboard
- a generic AI-generated website
- a PowerPoint clone
- a corporate template
- a minimal portfolio
- a childish 3D demo

The experience must feel:
- cinematic
- premium
- futuristic
- immersive
- expensive
- bold
- emotionally powerful

The UI must immediately create emotional impact within the first 10 seconds.

The entire experience should feel like:
- Apple product launches
- luxury brand campaigns
- high-budget commercial experiences
- Tesla-level visual confidence
- premium interactive showroom experiences

The project must feel handcrafted and art-directed.

No generic AI UI patterns.

Avoid:
- excessive gradients
- random glowing blobs
- cheap glassmorphism
- overused neon cyberpunk aesthetics
- dashboard cards everywhere
- template layouts
- childish animations
- too much text
- tiny typography

The UI should rely heavily on:
- scale
- spacing
- cinematic motion
- typography
- fullscreen visuals
- immersive transitions
- massive imagery
- bold layouts

Typography should be large, bold, dramatic, and spaced elegantly.

Text content should remain minimal.

The visuals should do most of the storytelling.

The application should feel alive.

The experience must contain:
- smooth scrolling
- cinematic transitions
- scroll-triggered animations
- fullscreen sections
- dynamic layout transformations
- layered motion
- immersive interactions
- premium pacing
- subtle randomness
- intelligent motion systems

The website should feel reactive to user movement and scrolling.

The project must use:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Lenis smooth scrolling
- Three.js / React Three Fiber

Three.js should be deeply integrated into the experience.

3D scenes must NOT feel like isolated demos.
They should feel embedded naturally into the storytelling.

The website should feel visually expensive.

AI-generated assets are allowed and encouraged.

If real assets are unavailable:
- automatically use cinematic AI-generated imagery
- use luxury-quality placeholder videos
- use premium editorial imagery
- use realistic 3D renders

The experience should prioritize:
1. emotional impact
2. visual sophistication
3. cinematic storytelling
4. motion quality
5. immersive interaction
6. premium presentation

Code quality must remain production-grade.

Maintain:
- reusable components
- modular architecture
- responsive layouts
- clean structure
- optimized rendering
- smooth performance

Every section should feel intentionally designed.

Nothing should feel generic.
